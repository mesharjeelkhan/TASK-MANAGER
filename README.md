# TASK-MANAGER
Task Manager built with the MERN stack (MongoDB, Express, React, Node.js).
Features include user authentication, task creation, updates, deletion, and real-time status management with a responsive UI.
A full-stack Task Manager application developed using the MERN stack.
Users can securely register and log in, create and manage tasks, update task status, and delete tasks. 
The project follows RESTful API principles, uses MongoDB for data storage, and provides a responsive React-based user interface.

A modern MERN Stack Task Manager application with:

User authentication (JWT)

CRUD operations for tasks

Task status tracking (Pending / In Progress / Completed)

Secure REST APIs with Node.js & Express

MongoDB for persistent storage

Responsive frontend built with React
